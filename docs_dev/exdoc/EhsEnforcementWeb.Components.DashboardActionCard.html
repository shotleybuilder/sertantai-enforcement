    EhsEnforcementWeb.Components.DashboardActionCard (ehs_enforcement v0.1.0)

    Base dashboard action card component with customizable slots for metrics, actions, and admin indicators.

    This component provides a reusable foundation for all dashboard action cards with:

    • Configurable themes (blue, yellow, purple, green)
    • Responsive 1x4 grid layout
    • Loading, error, and hover states
    • Admin privilege indicators
    • Accessibility features

    card_action_button(assigns)

    Renders a primary action button for cards.

    Examples

    <.card_action_button phx-click="browse_recent">
  Browse Recent
</.card_action_button>

    Attributes

    • class (:string) - Additional CSS classes. Defaults to "".
    • disabled (:boolean) - Whether button is disabled. Defaults to false.
    • Global attributes are accepted. Additional HTML attributes. Supports all globals plus: ["phx-click", "phx-value-*", "type", "href", "target"].

    Slots

    • inner_block (required) - Button content.

    card_secondary_button(assigns)

    Renders a secondary action button for cards (admin or less prominent actions).

    Examples

    <.card_secondary_button phx-click="search_cases">
  Search Cases
</.card_secondary_button>

    Attributes

    • class (:string) - Additional CSS classes. Defaults to "".
    • disabled (:boolean) - Whether button is disabled. Defaults to false.
    • admin_only (:boolean) - Whether this is an admin-only button. Defaults to false.
    • Global attributes are accepted. Additional HTML attributes. Supports all globals plus: ["phx-click", "phx-value-*", "type", "href", "target"].

    Slots

    • inner_block (required) - Button content.

    dashboard_action_card(assigns)

    Renders a dashboard action card with metrics, actions, and optional admin controls.

    Examples

    <.dashboard_action_card title="Cases Management" icon="üìÅ" theme="blue">
  <:metrics>
    <div class="text-2xl font-bold text-gray-900">1,003</div>
    <div class="text-sm text-gray-500">Total Cases</div>
  </:metrics>
  <:actions>
    <button class="btn-primary">Browse Recent</button>
  </:actions>
  <:admin_actions>
    <button class="btn-secondary">[ADMIN] Add New Case</button>
  </:admin_actions>
</.dashboard_action_card>

    Attributes

    • title (:string) (required) - Card title.
    • icon (:string) (required) - Card icon (emoji or HTML).
    • theme (:string) - Card theme: blue, yellow, purple, green. Defaults to "blue".
    • loading (:boolean) - Show loading state. Defaults to false.
    • error (:string) - Error message to display. Defaults to nil.
    • class (:string) - Additional CSS classes. Defaults to "".
    • Global attributes are accepted. Additional HTML attributes.

    Slots

    • metrics - Metrics display area. Accepts attributes:
      • class (:string) - Additional CSS classes for metrics.
    • actions - Primary action buttons. Accepts attributes:
      • class (:string) - Additional CSS classes for actions.
    • admin_actions - Admin-only action buttons. Accepts attributes:
      • class (:string) - Additional CSS classes for admin actions.
      • visible (:boolean) - Whether admin actions are visible.
