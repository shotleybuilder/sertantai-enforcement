    EhsEnforcementWeb.Components.NoticesActionCard (ehs_enforcement v0.1.0)

    Notices Management action card component for the dashboard.

    Displays notices metrics, provides filtered navigation, and admin-controlled create functionality. Implements the notices card specification from the dashboard action cards design document.

    Summary

    Renders the Notices Management action card with live metrics and actions.
