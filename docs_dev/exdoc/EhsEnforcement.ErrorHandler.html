    EhsEnforcement.ErrorHandler (ehs_enforcement v0.1.0)

    Comprehensive error handling system for the EHS Enforcement application.

    Provides error categorization, recovery strategies, circuit breaker patterns, error isolation, and comprehensive error monitoring and metrics.
