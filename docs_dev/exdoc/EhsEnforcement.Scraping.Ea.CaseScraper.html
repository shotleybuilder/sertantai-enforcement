    EhsEnforcement.Scraping.Ea.CaseScraper (ehs_enforcement v0.1.0)

    Environment Agency enforcement action scraper - Two-stage URI pattern.

    Stage 1: Single summary request (no pagination) gets ALL cases for date range Stage 2: Individual detail page requests with throttling between each request

    Rate limiting: Configurable delay between detail page requests (default 3s)
