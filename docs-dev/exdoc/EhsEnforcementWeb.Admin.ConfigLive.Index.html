    EhsEnforcementWeb.Admin.ConfigLive.Index (ehs_enforcement v0.1.0)

    Admin configuration management overview page.

    Displays all scraping configurations with their current status and provides navigation to detailed configuration management. Administrators can:

    • View all scraping configurations and their active status
    • Navigate to scraping configuration editor
    • Create default configurations if none exist
    • See feature flag summaries for each configuration

    Routes

    • GET /admin/config - Configuration overview page

    Authentication

    Requires admin authentication. Uses current_user from socket assigns for authorization of configuration access.

    Mounts the configuration overview page.

    Loads all scraping configurations synchronously and displays their status. Handles cases where current_user might be nil by falling back to unauthenticated reads.

    • Sets errors if configuration loading fails
